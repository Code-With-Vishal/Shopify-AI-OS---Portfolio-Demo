# Shopify AI OS

Shopify AI OS is a 3-agent automation dashboard featuring:
- Customer Support Bot
- Cart Recovery Specialist
- Product Description Generator

## Setup

1. Add your `OPENAI_API_KEY` to Vercel environment variables or in a `.env` file for local development.

## Deployment

Deploy this project on Vercel:

1. Push this code to a GitHub repository.
2. Import the project in Vercel.
3. Add `OPENAI_API_KEY` to the Environment Variables settings.
4. Deploy!
